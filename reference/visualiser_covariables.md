# Créer une visualisation synthétique des covariables extraites

Créer une visualisation synthétique des covariables extraites

## Usage

``` r
visualiser_covariables(
  resultats,
  type = c("tous", "terrain", "ombrage", "vent", "arbres", "distances", "consolide")
)
```

## Arguments

- resultats:

  Liste retournée par extraire_covariables_complet()

- type:

  Type de visualisation: "terrain", "ombrage", "vent", "arbres",
  "distances", "consolide", ou "tous"

## Value

NULL (affiche les graphiques)
