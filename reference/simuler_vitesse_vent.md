# Simuler la vitesse du vent

Simuler la vitesse du vent

Simuler la vitesse du vent avec effet de fetch

## Usage

``` r
simuler_vitesse_vent(fetch, vitesse_ref = 5, coef_acceleration = 0.3)

simuler_vitesse_vent(fetch, vitesse_ref = 5, coef_acceleration = 0.3)
```

## Arguments

- fetch:

  Résultat de calculer_fetch_vent()

- vitesse_ref:

  Vitesse de référence (m/s)

- coef_acceleration:

  Coefficient d'accélération après les arbres

- result:

  Résultat de calculer_distances_amont_aval()

- coef_amont:

  Effet ralentissement amont

- coef_aval:

  Effet accélération aval
