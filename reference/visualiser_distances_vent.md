# Visualiser les distances amont/aval

Visualiser les distances amont/aval

## Usage

``` r
visualiser_distances_vent(
  result,
  type = c("comparaison", "amont", "aval", "totale")
)
```

## Arguments

- result:

  Résultat de calculer_distances_amont_aval()

- type:

  "amont", "aval", "totale" ou "comparaison"

## Value

ggplot2
