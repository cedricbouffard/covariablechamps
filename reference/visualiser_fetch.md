# Visualiser le fetch de vent

Visualiser le fetch de vent

## Usage

``` r
visualiser_fetch(
  fetch,
  type = c("elliptique", "simple", "comparaison"),
  titre = NULL
)
```

## Arguments

- fetch:

  Résultat de calculer_fetch_vent()

- type:

  "simple", "elliptique", ou "comparaison"

- titre:

  Titre optionnel

## Value

ggplot2
