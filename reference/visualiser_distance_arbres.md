# Visualiser la distance aux arbres

Visualiser la distance aux arbres

## Usage

``` r
visualiser_distance_arbres(
  result,
  type = c("buffer", "points", "comparaison"),
  titre = NULL
)
```

## Arguments

- result:

  Résultat de calculer_distance_arbres()

- type:

  "buffer", "points" ou "comparaison"

- titre:

  Titre optionnel

## Value

ggplot2
